# Smoking Health Risk Analysis Dashboard

## Overview
This project is an interactive Power BI dashboard designed to analyze smoking-related health risks using healthcare data.

The dashboard explores:
- Smoking status
- Cholesterol levels
- BMI
- Age groups
- Gender-based smoking trends
- Years of Smoking (YOS)
- Cigarettes Per Day (CPD)

## Objective
The main objective of this project is to understand how smoking habits affect patient health and identify high-risk groups based on age, gender, BMI, and cholesterol levels.

## Dataset Information
The dataset contains healthcare-related patient information such as:
- Age
- Gender
- BMI
- Smoking Status
- Cholesterol Level
- Years of Smoking
- Cigarettes Per Day

## KPIs Used
- Total Patients
- Average Age
- Average BMI
- Average CPD
- Average YOS

## Dashboard Features
- Smoking status distribution using donut chart
- Gender-wise smoking analysis using Sankey chart
- Cholesterol level analysis
- Years of Smoking and Cigarettes Per Day by age group
- Interactive filters and slicers

## Key Insights
- Most patients are non-smokers
- Male patients show a higher smoking tendency
- Middle-aged groups have higher smoking duration
- Smoking may be linked with higher cholesterol levels
- Older patients may reduce smoking due to health concerns

## Tools and Technologies
- Power BI
- Power Query
- DAX
- Excel / CSV
