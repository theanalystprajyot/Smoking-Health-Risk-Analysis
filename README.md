# Smoking-Health-Risk-Analysis
Interactive Power BI dashboard for analyzing smoking-related health risks using healthcare data. The project explores smoking status, cholesterol levels, BMI, age groups, gender trends, years of smoking, and cigarettes per day to generate meaningful patient health insights.
Tools Used
- Power BI
- Power Query
- DAX
- Excel / CSV

## Features
- Smoking status distribution
- BMI analysis
- Cholesterol analysis
- Age group analysis
- Gender-wise smoking trends
